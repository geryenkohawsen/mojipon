# Mojipon

Mojipon is a Japanese-aware Slack emoji pack generator.

It helps teams create Slack-ready emoji packs with multiple filename conventions:

- Romaji: `nandesuka.png`
- Kana: `なんですか.png`
- Original Japanese: `何ですか.png`

## Core idea

Most emoji generators create one image with one filename.

Mojipon creates workspace-ready emoji packs for Japanese and global Slack teams.

## Planned features

- Emoji text/image generator
- Romaji, kana, and original Japanese filename variants
- ZIP export
- Batch generation
- Slack-safe filename validation
- Emoji pack library
- Creator-uploaded emoji bundles
- Free and paid creator packs
- Workspace naming convention presets

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- PWA-ready architecture
